## 🪾 Descripción 

**Celestial IA** es un bot de WhatsApp basado en `@whiskeysockets/baileys` que **responde automáticamente a todos tus mensajes** mediante inteligencia artificial (sin necesidad de prefijos). Está programado para mantener conversaciones fluidas, actuar como el personaje que elijas y nunca revelar que es una IA. Todo ello con una **vinculación segura mediante código de 8 dígitos** (sin QR).

---

## 🥦 Características

- 🤖 **IA sin prefijo** – Responde a cualquier mensaje de texto automáticamente.
- 🔐 **Vinculación por código de emparejamiento** – Sin complicados códigos QR.
- 🎭 **Personalidad adaptable** – Actúa como el personaje que le indiques (por defecto: Momo Ayase de Dan Da Dan).
- 🗣️ **Conversación natural** – Usa el nombre del usuario cuando sea necesario.
- 🔄 **Reconexión automática** – Vuelve a conectarse si se cae la conexión.
- ☁️ **Listo para la nube** – Compatible con Render, Koyeb, Railway y cualquier PaaS.

## 🚀 Instalación y despliegue

### 🍒 En la nube (Recomendado)

<details>
<summary><strong>Render / Koyeb / Railway</strong> — Usando GitHub</summary>

```bash
git clone https://github.com/SergioOficial/Celestial-AI.git
cd Celestial-AI
npm install
npm start

</details>

🍒 En Termux (Android)

<details>
<summary><strong>Instalación paso a paso</strong></summary>

```
bash
termux-setup-storage
apt update && apt upgrade -y
pkg install -y git nodejs ffmpeg imagemagick
git clone https://github.com/SergioOficial/Celestial-AI.git
cd Celestial-AI
npm install
npm start
```
</details>

🧠 Comandos útiles para mantener activo el bot en Termux

<details>
<summary><strong>Usar PM2 (gestor de procesos)</strong></summary>

```bash
npm i -g pm2
pm2 start index.js --name "celestial"
pm2 save
pm2 logs
```

· Detener: pm2 stop celestial
· Reiniciar: pm2 start celestial
· Ver logs en segundo plano: pm2 logs celestial
· Eliminar proceso: pm2 delete celestial

</details>
